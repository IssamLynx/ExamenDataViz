Sources et outils

Source de données

Il s'agit d'une exploration des données de The Movie Database (TMDB) de 1910 à 2016. Un film à succès commercial ne fait pas que divertir le public, il permet également aux réalisateurs de générer des profits importants. Plusieurs facteurs tels que de bons réalisateurs, des acteurs expérimentés et le moment de la sortie sont importants pour la rentabilité, mais ils ne garantissent pas toujours l'évaluation d'un film. TMDB

Bokeh

Bokeh est une bibliothèque Python permettant de créer des visualisations interactives pour les navigateurs Web modernes. Elle vous aide à créer de magnifiques graphiques, allant de simples tracés à des tableaux de bord complexes avec des ensembles de données en continu. Avec Bokeh, vous pouvez créer des visualisations alimentées par JavaScript sans avoir à écrire vous-même le moindre JavaScript. Bokeh

chart studio

Cette librairie contient des utilitaires pour l'interface avec le service Chart Studio de Plotly (à la fois Chart Studio cloud et Chart Studio On-Prem). Avant la version 4 de plotly.py, cette fonctionnalité était incluse dans le paquetage plotly sous le module plotly.plotly. Dans le cadre de la version 4 de plotly.py, la fonctionnalité Chart Studio a été retirée du paquet plotly et publiée dans le paquet chart-studio. chart-studio
